# 训练脚本说明

实际脚本: `train_model.py`

## 用法
```bash
python train_model.py
```

## 功能
1. 加载 `data/` 目录下的所有 CSV 文件
2. 数据预处理和窗口切分
3. 特征提取 (mean, std, energy)
4. 训练 Logistic Regression 模型
5. 评估并输出混淆矩阵
6. 自动导出参数到 `Inc/model_params.h`

## 参数配置
- WIN_SEC = 0.5 (窗口长度秒)
- HOP_SEC = 0.1 (滑动步长秒)
- FS = 50 (采样率Hz)
