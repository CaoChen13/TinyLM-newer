# 模型参数说明

实际文件: `Inc/model_params.h`

## 模型信息
- 类型: Logistic Regression (逻辑回归)
- 特征维度: 18 (6通道 × 3特征)
- 类别数: 6

## 参数
- 窗口: 0.5秒 (25 samples @ 50Hz)
- 步长: 0.1秒 (5 samples)
- 重叠率: 80%

## 特征提取
每个通道计算3个特征:
- mean (均值)
- std (标准差)
- energy (能量/均方值)

## 预测公式
```
logits[k] = sum(features[i] * LR_COEF[k*18 + i]) + LR_INTERCEPT[k]
pred = argmax(logits)
```
